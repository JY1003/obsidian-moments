# Obsidian Moments (我的碎片化生活记录)

![Obsidian](https://img.shields.io/badge/Obsidian-Plugin-purple) ![Version](https://img.shields.io/badge/Version-1.0.0-blue)

## 💡 开发初衷：为什么我们需要一个“本地朋友圈”？

在这个社交网络泛滥的时代，你是否发现自己越来越不愿意发朋友圈了？每次满怀热情地想分享生活的一个瞬间，一打开微信，想到通讯录里成百上千个半生不熟的同事、客户和点头之交，光是斟酌文案、设置分组屏蔽，就耗尽了所有的分享欲。最终，那张照片只能默默躺在相册里吃灰。

但是，记录生活本身是极具意义的。想象一下，几十年甚至百年之后，当我们（或者我们的后代）能够再次回望这一生留下的生活碎片——那些去过的山川湖海、吃过的街边小店、突发奇想的碎碎念……那将是多么珍贵的财富。

既然不想发给别人看，那就纯粹地记录给自己看。这就是 `Obsidian Moments` 诞生的初衷：为你打造一个完全私有、数据永远掌握在自己手里、且拥有极致九宫格视觉体验的“本地朋友圈”。

## ✨ 核心特性

* 📸 **智能九宫格排版**：上传多张图片会自动根据数量（1张、2-4张、5-9张）完美适配等比裁切的网格布局，高度复刻经典视觉体验。
* 🎨 **原生极简美学**：抛弃社交软件多余的“假头像”和笨重的按钮，采用克制的纯文本交互和浅灰气泡，深度融入 Obsidian 原生阅读体验。
* 💬 **纯净的评论系统**：告别臃肿的 YAML 属性！你的评论会自动以 Markdown 列表的形式追加在源文件最底部的 `### 评论区`，保持笔记干净清爽。
* 🔒 **数据绝对私有**：没有云端数据库，没有隐私泄露。每一条动态都是一个标准的本地 `.md` 文件和本地图片。

## 🚀 如何使用？(Quick Start)

### 第一步：插件配置
安装并启用插件后，进入设置页面，填写以下基础配置：
* **Moments 存储路径**：你希望存放所有动态 `.md` 文件的文件夹（例如：`1.个人/朋友圈/`）。
* **附件存储路径**：动态中上传的图片存放位置（例如：`1.个人/朋友圈/图片/`）。
* **展示顺序**：选择按时间“正序”或“倒序”展示。

### 第二步：发布你的生活碎片
1. 随时随地唤出 Obsidian 命令面板（`Ctrl/Cmd + P` 或移动端下拉）。
2. 输入并执行命令：`创建 Moments`。
3. 在弹出的沉浸式面板中输入标题、📍地点、正文，并一键多选上传照片，点击“发布”。
4. 在 iOS 上，地点输入框右侧会额外显示 **「地图选址」** 按钮，可直接打开 Apple 地图；如果地点框里已经填了内容，按钮会优先用当前内容在地图中打开/搜索。
5. 地点字段支持直接填写 Markdown 链接，例如：`[Apple Park](https://maps.apple.com/?q=Apple%20Park)`；渲染 Moments 时会显示为可点击链接。

### 第三步：渲染瀑布流视图
在你的 Obsidian 库中新建一个笔记（例如命名为 `我的朋友圈.md`），在正文中输入以下代码块：

```markdown
` ` `moments
` ` `
```

切换到阅读模式或实时预览模式，这个代码块就会自动抓取你的记录，渲染出精美的瀑布流！

## 📂 透明的数据结构

Moments 拒绝任何私有格式绑定。你创建的每一条记录，在本地看起来都是这样极其标准的 Markdown：

```markdown
---
标题: 姑娘山一日游
地点: "[四姑娘山镇](https://maps.apple.com/?q=%E5%9B%9B%E5%A7%91%E5%A8%98%E5%B1%B1%E9%95%87)"
创建时间: 2026-03-01 12:47:14
更新时间: 2026-03-01 14:27:00
---

## 正文
太棒了从来没有这样体验过🍒

## 图片
![[IMG-20260301-124714.png]]
![[IMG-20260301-124715.png]]

```

## 📦 手动安装指南
1. 前往本仓库的 Releases 页面下载最新版本的压缩包。
2. 解压后，将 `main.js`、`manifest.json` 和 `styles.css` 放入你 Obsidian 库的 `.obsidian/plugins/obsidian-moments/` 文件夹中。
3. 确保你的 Obsidian 版本不低于 `1.11.0`。
4. 重启 Obsidian，在“第三方插件”设置中启用 `Moments`。
